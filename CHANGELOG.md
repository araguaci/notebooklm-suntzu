## [1.0.0] - 2024-01-XX

### Adicionado
- Interface moderna com design dark mode
- Sistema de gerenciamento dinâmico de materiais via JSON
- Suporte para múltiplos tipos de mídia (documentos, vídeos, áudios, imagens)
- Script interativo para adicionar materiais
- Players integrados para vídeo, áudio e imagens
- Sistema de tags para categorização
- Links diretos para NotebookLM
- Animações suaves com Framer Motion
- Layout totalmente responsivo
- Documentação completa
- Guias passo a passo
- Templates para GitHub Issues
- CI/CD com GitHub Actions

- Guia completo de adição de materiais
