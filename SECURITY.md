# Política de Segurança

## Versões Suportadas

Atualmente, estamos dando suporte às seguintes versões:

| Versão | Suportada          |
| ------ | ------------------ |
| 1.0.x  | :white_check_mark: |

## Reportando uma Vulnerabilidade

Se você descobriu uma vulnerabilidade de segurança, por favor:

1. **NÃO** abra uma issue pública
2. Envie um email para: seu-email@exemplo.com
3. Inclua:
   - Descrição da vulnerabilidade
   - Passos para reproduzir
   - Impacto potencial
   - Sugestões de correção (se houver)

Vamos responder dentro de 48 horas e trabalhar para corrigir a vulnerabilidade o mais rápido possível.

## Boas Práticas de Segurança

- Mantenha suas dependências atualizadas
- Não commite credenciais ou informações sensíveis
- Use HTTPS para todas as conexões
- Valide todos os inputs do usuário
- Mantenha o Node.js e npm atualizados

## Agradecimentos

Agradecemos a todos que reportam vulnerabilidades de forma responsável!
